```
📱 你：按住话筒 🎤 说话（支持中英文混说）
    │
    ▼
🧠 AI 听到你说的 → 分析你的英语水平（A1-C2）

    │
    ▼
```

**打字 + 语音输入 + 语音播报 + 翻译**，四大模式无缝切换。这才是真正的沉浸式语言环境。

## ❓ FAQ

<details>
<summary><b>Q: 真的完全免费吗？</b></summary>

App 本身 MIT 协议开源，永久免费。对话需要 DeepSeek API Key（充 5 元用几个月）。DeepSeek 的价格是 OpenAI 的 1/50，极其便宜。
</details>

<details>
<summary><b>Q: AI 聊天跟真人聊天最大的区别是什么？</b></summary>

最大的区别是**不丢脸**。你跟真人说话会紧张、怕语法错、怕单词忘，大脑直接宕机。跟 AI 说话没这个心理负担——你说得再烂，它都秒回。正是这种"零压力"让你敢开口，开口多了自然就流利了。
</details>

<details>
<summary><b>Q: 零基础能用吗？</b></summary>

能用。内置的 A1-C2 水平检测会自动评估你的水平，AI 会根据你的水平调整回复的词汇难度。从零开始每天聊 15 分钟，1 个月能到 A2 水平。
</details>

<details>
<summary><b>Q: 聊天内容有记录吗？</b></summary>

有。最近 50 条消息保存在你浏览器的 localStorage 里（不上传到任何服务器）。你可以回头看你跟每个角色聊了什么，追踪你的进步。
</details>

<details>
<summary><b>Q: 我的 API Key 安全吗？</b></summary>

API Key 存储在你的浏览器 localStorage 中，只在你调用 DeepSeek API 时发送给 DeepSeek 服务器。不会经过任何第三方服务器。
</details>

<details>
<summary><b>Q: 能用自己的 OpenAI Key 吗？</b></summary>

可以。启动页面上有个 Base URL 输入框，换成 OpenAI 的 API 地址即可。所有 OpenAI 兼容的 API 都能用（DeepSeek / OpenAI / Azure / 本地模型等）。
</details>
